Colorful theme for Visual Studio Code

Install

Install using Command Palette

  • Go to View -> Command Palette or press
  • Ctrl+Shift+P - Then enter Install Extension - Write Cotton Candy
  • Select it or press Enter to install

Install from source 

$ git clone https://github.com/forgng/Cotton-Candy.git ~/.vscode/extensions/theme-cotton-candy
